Composer
=========

Ansible role for installing composer. Tested platforms are:
* Debian 9
* Ubuntu 16

Requirements
------------

Debian 9 (stretch)
Ubuntu 16 (xenial)

Role Variables
--------------

None

Dependencies
------------

None

Example 
----------------
    ---
    - hosts: all
      roles:
         - { role: antonchernik.composer }
